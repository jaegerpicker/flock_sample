### Shawn Campbell's Flock Javascript sample

This is a sample written from node.js 5.6+ on Mac OS X

## Requirements:
node
npm
jest
babel
ramda
yarn (optional)

## Setup: 
install node:

brew install node

install jest:
npm install jest -g

install yarn:
npm install yarn -g

## To run the sample:
git clone the repo
From the command line
yarn install (npm install)

# To run the test suite:
npm test
